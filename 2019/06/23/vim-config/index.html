vim配置C/C++阅读、开发神器

作者 王振华 日期 2019-06-23
vim配置C/C++阅读、开发神器

Linux下没有类似于source insight这样方便快捷的工具，但是vim一样强大，我们可以把vim配置成类似于source insight一样的C/C++开发工具，让我们阅读、编辑代码，事半功倍！

1、安装Vundle

Vundle是vim的一个插件管理工具，非常好用，我们可以使用Vundle快速安装我们的插件。安装Vundle可根据这个链接来操作https://github.com/VundleVim/Vundle.vim。

4、安装高亮（mark）插件

source insight很吸引我的一个功能是可以高亮单词，这样在阅读代码时将关键字高亮出来更容易查看。那配置vim类似于source insight自然高亮单词这个功能不能缺少。
从https://github.com/wangzhenhua92429/my_vimrc下载mark.vim放到/usr/share/vim/vim73/plugin/目录下。阅读代码时，选中单词点击”\ + m”即可高亮单词，“\ + n”取消高亮。
